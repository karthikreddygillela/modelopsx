OneFix Architecture (Plug & Play Flow)

1️⃣ Use Case Onboarding

When a use case is onboarded, necessary automation assets are created in the background.

Splunk JSON Generator
Deployment Script Creator
Metadata Collector
⬇

2️⃣ Deployment Phase

Deployment triggers across various platforms based on configuration.

Harness
IBM UCD
Ansible Tower
CloudRun/K8s
⬇

3️⃣ Monitoring & Error Capture

Dashboards fetch logs from Splunk and update our internal UI in real-time.

Splunk Dashboards
Internal Monitoring UI
Slack/Email Alerts
⬇

4️⃣ OneFix Error Registry

Known/resurfaced errors are auto-fixed. New errors are stored with fix notes by engineers.

Error → Fix Mapping
Manual Fix Entry (Human-in-Loop)
OneFix Central DB
⬇

5️⃣ Agent-Driven Auto Healing

Agents trained on past fixes take automatic actions on familiar errors.

AI Agent Executes Fix
Feedback Loop
Improved Error Intelligence
