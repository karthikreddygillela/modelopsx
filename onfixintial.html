OneFix Principle Architecture

📦 Usercase Onboarding
Create Splunk JSON, deployment scripts via API
⚙️ Deployment Phase
Targets: Harness, IBM UCD, Ansible
Scripts stored, triggered
📊 Monitoring Dashboard
Data fetched from Splunk to main UI
📋 OneFix Error Board
Known/resurfaced errors → fixed automatically
New errors → suggested to human-in-loop
🤖 AI Agents
Gradually take over fixing known issues
Improves via feedback over time
